# Project 3: Continual Learning for Multi-Cuisine Robotics

## Vision
Enable the humanoid robot to learn new cuisines (Italian, Chinese, Mexican) without forgetting Indian cooking.

## Approach
1. **Elastic Weight Consolidation (EWC)** - Preserve important weights
2. **Replay Buffers** - Maintain exemplars from previous cuisines
3. **Progressive Neural Networks** - Add new modules per cuisine
